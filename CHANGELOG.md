# Changelog

All notable changes to RocketRAM will be documented in this file.

## [1.0.0] - 2025-12-28

### Added
- Initial release of RocketRAM
- Real-time system monitoring (CPU, RAM, Disk, Network, GPU, Battery)
- Process management and termination
- System cleanup (temp files, cache, logs, recycle bin)
- System optimization (startup programs, services, disk)
- Disk space analyzer
- Network connection monitoring
- Battery optimization
- System health scoring
- Scheduled cleanup and optimization
- Smart alerts with configurable thresholds
- Desktop notifications
- Settings panel with comprehensive options
- Beautiful dark theme UI with cyberpunk-inspired design
- Cross-platform support (Windows, macOS, Linux)

### Features
- **CPU Monitoring:** Real-time CPU usage, speed, temperature, core count
- **RAM Monitoring:** Memory usage, available memory, cached memory
- **Disk Monitoring:** Disk usage, read/write speeds, I/O stats
- **Network Monitoring:** Active interface info, download/upload speeds, network details
- **GPU Monitoring:** GPU usage, temperature, VRAM, clock speeds
- **Battery Monitoring:** Charge level, status, time remaining, health
- **Process Management:** View, search, filter, and terminate processes
- **System Cleanup:** Remove temporary files, cache, logs, and more
- **Optimization:** Optimize startup programs, services, and system performance
- **Notifications:** Get notified about cleaning, optimization, and system alerts
- **Health Scoring:** Get system health score with actionable recommendations

### Technical
- Built with Electron 28.0
- Uses systeminformation library for system monitoring
- Native Windows/macOS/Linux APIs for system operations
- Efficient real-time updates with minimal resource usage

- Comprehensive README
- Terms of Service

- Third-Party Licenses
- Installation and usage guides
